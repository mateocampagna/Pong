# Pong Game

![Logo](https://i.blogs.es/e4c828/pong/1366_521.png)

This project is a recreation of the classic Pong game, developed for educational purposes using C and raylib.

## Features

- Two-player paddle controls (keyboard input)
- Ball movement based on velocity vectors
- Collision detection with paddles and screen boundaries
- Score tracking system
- Simple and minimal game loop structure

## Concepts Practiced

- Real-time input handling
- Game loop design
- Basic physics simulation (position + velocity)
- Collision detection (AABB-style logic)
- Coordinate systems and screen boundaries

## Technologies

- C
- raylib
